# AiMiniGames

一个基于 Unity 制作的经典小游戏合集项目。

这个项目的目标，是尽可能用 AI 辅助完成小游戏开发流程，包括代码生成、结构搭建、美术素材准备与界面迭代；开发者主要负责 Unity 内的场景搭建、组件挂载、参数调整与功能测试。

## 项目内容

当前项目已包含 3 个可游玩的小游戏：

- `2048`
- `Match3` 三消
- `合成大西瓜`

项目同时提供了统一的 `MainMenu` 主菜单，可从首页进入各个小游戏场景。

## AI 辅助工作流

本项目采用了明确的 AI 协作方式：

- 使用 `Codex` 辅助脚本编写、功能迭代、项目结构整理与问题排查
- 使用 AI 图像工具辅助生成部分美术素材或视觉草稿
- 在 Unity 中手动完成 UI 摆放、预制体挂载、参数配置与实际测试

## 目录结构

- `Assets/Scenes`
  - `MainMenu.unity`：主菜单场景
- `Assets/_Project`
  - 项目级共享脚本、预制体、UI 与公共资源
- `Assets/Games/Game2048`
  - `2048` 相关场景、脚本与资源
- `Assets/Games/GameMatch3`
  - 三消相关场景、脚本与资源
- `Assets/Games/GameWatermelon`
  - 合成大西瓜相关场景、脚本与资源

## 开发环境

- Unity `2023.2.20f1c1`
- Git + GitHub
- VSCode + Codex 协作开发

## 项目说明

- 工程启用了 Unity 文本序列化与 `.meta` 文件管理，便于使用 Git 跟踪版本
- 当前版本以玩法原型、交互反馈与整体可玩性为主
- 后续仍可继续补充更完整的美术资源、音效、动画与打包发布流程
