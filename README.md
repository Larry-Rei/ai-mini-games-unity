# AiMiniGames

一个使用 Unity 制作的经典小游戏合集项目。

项目目标是尽量采用 AI 辅助工作流完成开发：

- 使用 Codex 辅助脚本编写、结构设计与代码迭代
- 使用 AI 工具生成美术素材与部分文案内容
- 在 Unity 中手动完成拖拽 UI、挂载脚本、参数配置与测试

## 当前规划

首个完整实现的游戏为 `2048`，在流程跑通后，再继续扩展其他经典小游戏。

当前预留的游戏方向包括：

- `2048`
- `消除方块`
- `合成大西瓜`

## 目录结构

- `Assets/_Project`：项目级共享资源，如通用脚本、UI、场景、预制体、音频与美术资源
- `Assets/Games/Game2048`：`2048` 专用资源与脚本
- `Assets/Games/GameBlockPuzzle`：`消除方块` 专用资源与脚本
- `Assets/Games/GameWatermelon`：`合成大西瓜` 专用资源与脚本

## 开发说明

- Unity 工程采用文本序列化与可见 `.meta` 文件，便于 Git 版本管理
- 仓库会优先保持清晰的项目结构，方便后续逐步扩展多个小游戏
